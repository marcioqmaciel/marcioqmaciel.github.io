Jekyll no Bash no Ubuntu no Windows!

Há várias formas de se usar Linux e Windows, com vantagens e desvantagens, como vários PCs na rede, dual boot, VM, docker e agora o Bash On Windows. O Bash On Windows é uma forma conveniente para desenvolvedores Windows usarem a shell do Linux diretamente no Windows, mas não se esqueça de que ainda está em versão beta com limitações e problemas. O nosso objetivo com esse tutorial é executar o Jekyll, um web server de arquivos estáticos, feito em Ruby, ideal para criar blogs leves sem a necessidade de usar bases de dados. O GitHub oferece a hospedagem gratuita de web sites estáticos e o Jekyll é o serviço por trás dessa hospedagem. Neste tutorial estou supondo que você é usuário do Windows que queira aproveitar algumas funcionalidades exclusivas do Linux ou que sejam de difícil instalação diretamente no Windows.

Instalando o Linux no Windows

Se você já instalou o bash no Windows, pode pular para a próxima etapa. Caso ainda não tenha instalado, deve primeiro preparar seu Windows para receber o bash. Para que possa comparar, estou executando o Windows 10 Insiders Preview build 14971. Primeiro, vá em Configurações > Atualização e Segurança > Para desenvolvedores e marque a opção Modo de desenvolvedor. Agora, digite ativar na Cortana e selecione Ativar ou desativar recursos do Windows. Na janela que se abre selecione Subsistema do Windows para Linux (Beta). O Windows fará a instalação e reiniciará o PC. Após reiniciar, abra uma janela do PowerShell como administrador (clique com o botão direito no logo do Windows, na build 14971 a opção de prompt de Command foi substituída por PowerShell). No PowerShell execute o comando lxrun /install. Neste momento o Windows irá baixar (tenha paciência, o processo é demorado) uma imagem da distro Ubuntu 16.04.1 LTS e a instalará dentro do WSL - Windows Subsystem for Linux. Sim, sem a necessidade de uma pesada VM, você estará instalando e executando uma distro do Linux dentro do Windows 10! Durante a instalação o PowerShell perguntará pelo nome de usuário e senha que serão usados internamente pela distro. Ao término da instalação, feche o PowerShell e abra o Bash no Ubuntu no Windows no modo administrador, clicando no atalho criado no menu Iniciar.

Preparando o Linux

Com o Linux instalado no Windows, vamos atualizá-lo e prepará-lo para o Jekyll (Não se preocupe se você não está familiarizado com os comandos a seguir. Se precisar pesquisar mais informações pela internet, lembre-se de que a distro instalada é a Ubuntu, então os comandos do bash seguem o padrão dessa distro). A primeira coisa a fazer é atualizar o Linux, digitando sudo apt update. O bash irá lhe pedir a senha de usuário criada na instalação, pesquisará e atualizará a base do Ubuntu. Aguarde, o processo pode demorar vários minutos. Após essa primeira atualização, o bash pode lhe retornar que há mais atualizações secundárias disponíveis (no meu caso eram 85 pacotes!). Execute sudo apt full-upgrade para atualizar todos os pacotes, e novamente aguarde o processo de download e instalação. Após a atualização, se você quiser pode executar apt list --upgradable para ver se ainda resta algum pacote desatualizado.
